4.. What is a callback function and why will you use it?


A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.


A callback's purpose is to execute code in response to an event. These events can be like mouse clicks; with callback, we can add text at the end of each statement like"execute this code every time the user clicks a key on the keyboard."
